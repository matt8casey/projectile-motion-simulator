# Governing Equations

## Horizontal Position

x = v₀ cos(θ)t

## Vertical Position

y = v₀ sin(θ)t − ½gt²

## Horizontal Velocity

vx = v₀ cos(θ)

## Vertical Velocity

vy = v₀ sin(θ) − gt

## Drag Force

Fd = ½ρCdAv²

where:

ρ = air density
Cd = drag coefficient
A = cross-sectional area
v = velocity
